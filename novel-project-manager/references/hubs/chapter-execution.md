# Hub: Chapter Execution

适用任务：

- 继续写下一章
- 单章开写前做方法论准备
- 场景拆分、章节推进、章尾钩子
- 补场面、补细节、补画面感、去悬浮

先读顺序：

1. `references/chapter/chapter-workflow.md`
2. `references/chapter/chapter-guide.md`
3. `references/chapter/writing-quickref.md`

按需下钻：

- 正文约束模板：`references/chapter/chapter-template.md`
- 开头 / 钩子：`references/chapter/hook-techniques.md`
- 文学感开篇：`references/chapter/literary-opening.md`
- 分类写法 / 去悬浮：`references/chapter/descriptive-taxonomy.md`
- 对白：`references/chapter/dialogue-writing.md`
- 群像：`references/chapter/ensemble-writing.md`
- 流断节奏：`references/chapter/flow-break-writing.md`
- 扩写：`references/chapter/content-expansion.md`
- 日常叙事：`references/chapter/daily-narrative.md`
- 非线性结构：`references/chapter/nonlinear-narrative.md`
- 读者补偿：`references/chapter/reader-compensation.md`
- 悬念设计：`references/chapter/suspense-design.md`

默认只读能直接影响这一章落笔的 2-4 份，不要把整套创作方法都塞进当前上下文。
