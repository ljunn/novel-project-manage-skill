# 纯正文章节模板

`manuscript/` 下的章节文件默认只保留小说正文本身，不写写作辅助壳。

不要输出以下内容：

- `# 第[X]章：[章节标题]`
- `## 本章概要`
- `## 正文`
- `## 章节备注`
- `---`

章节编号和标题放在文件名中，摘要、伏笔、下章预告写入 `task_log.md`、`docs/章节规划.md` 或 `plot/伏笔记录.md`。

平台向快节奏章节可选：

- 动作、冲突、追逐、争吵、强情绪场优先短流段推进
- 每段尽量只写一个动作、一个反应、一个画面或一句对白
- 段尾尽量停在动作未完、风险逼近、真相露角或情绪拐点
- 但全文仍要连着气，不能切成互不相干的碎片

写入格式：

```text
[从第一句正文直接开始]

[中段持续推进冲突、回应旧悬念、制造新问题]

[结尾停在能驱动下一章的动作、发现、选择或危险上]
```
