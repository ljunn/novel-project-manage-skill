# 分类写法卡

用于处理下面这些高发问题：

- 场面发干，只有剧情提要，没有人在场
- 想去 AI 味，却只会删套语，不会补观察点
- 想补细节、画面感、质感、压迫感，但越写越像词藻堆砌
- 返修时知道“这段悬浮”，却不知道该从哪一类描写下手

核心原则：

- 先判断当前段落主要在写 `人 / 景 / 物 / 事` 哪一类
- 每段只抓 `1 个主体 + 2 个观察维度 + 1 个变化/结果`
- 词库只拿来提示观察方向，不拿来堆砌成排形容词
- 一切描写都要服务 POV、冲突、关系、代价或节奏，不能脱离当前场景单独抒情

## 最短调用法

1. 先判类：这段主要是在写人、写景、写物，还是写事
2. 再选维度：只选 2 个最有用的观察维度
3. 再补变化：让细节进入动作、阻力、反应或后果
4. 最后复查：删掉可以互换到任何书里的空泛形容

## 1. 写人

常用维度：

- 肖像：脸、眼、嘴、头发、衣着、站姿、伤痕、疲态
- 言谈：语速、停顿、口头禅、回避、顶撞、试探、答非所问
- 动作：手怎么放、肩背怎么绷、脚步怎么停、怎样碰环境和物件
- 小习惯：摸烟、抹袖口、盯门、掐指节、咬字、避视线
- 关系压力：他在谁面前收着、硬着、装着、躲着、讨好着

落笔提醒：

- 不要先写“英俊 / 美丽 / 冷艳 / 威严”，先写能让 POV 立刻注意到的差异点
- 人物气质优先从动作和言谈里出，不从作者评语里出
- 写人物时，优先让外貌、动作和关系状态互相咬合

返修提示：

- 如果人物像纸片人，别急着补设定，先补一个高识别动作和一个关系反应
- 如果人物都说同一种话，优先改语速、停顿、遮掩方式和信息密度

## 2. 写景

常用维度：

- 天光：亮度、颜色、方向、被遮挡还是直照
- 气象：风、雨、雾、热、冷、闷、潮、干
- 声音：脚步、雨棚、远处喇叭、门轴、虫鸣、人群、机器
- 气味：潮味、土腥、药味、油烟、血气、霉味、铁锈味
- 地表与障碍：泥、水、台阶、栏杆、墙角、路滑、视线死角、可藏身处
- 场面密度：空、堵、挤、散、乱、亮、暗、脏、净

落笔提醒：

- 景不是壁纸，而是人物行动时会碰上的压力、掩护、阻碍或反差
- 优先写会影响动作和判断的环境因素，而不是平均铺陈“很美 / 很冷 / 很荒凉”
- 同一个景物，最好带一个动态变化：起风、变暗、积水漫上来、灯闪了一下

返修提示：

- 如果场景悬浮，先补“人物脚下是什么、手边有什么、耳边在响什么”
- 如果气氛太散文化，删掉抒情判断，改成环境怎样干扰了人

## 3. 写物

常用维度：

- 材质：铁、木、布、皮、瓷、玉、纸、塑料、玻璃
- 体量：长短、轻重、厚薄、粗细、占不占地方
- 状态：新旧、脏净、裂损、卷边、掉漆、发潮、变形、发烫
- 使用痕迹：磨亮、缺口、汗渍、油污、折痕、补丁、血点
- 归属感：谁常用、谁珍惜、谁嫌弃、谁拿来当武器或遮掩
- 功能压力：能不能藏、能不能砸、会不会响、会不会暴露身份

落笔提醒：

- 物件优先写用途、损耗和归属，不要只写“华美 / 古朴 / 精致”
- 写物最好带人：谁拿它，谁盯它，谁因为它受限或获利
- 同一件物，可以同时承担情绪锚点、关系信号或伏笔功能

返修提示：

- 如果道具像布景板，补上“它被怎样使用”
- 如果物件没有记忆点，补一个可识别缺损或异常用法

## 4. 写事

常用维度：

- 起手动作：谁先动，怎么动，目标是什么
- 受阻：被谁拦、被什么卡、因为什么误判
- 变招：改口、转身、换路线、抬价、抢先、示弱、下狠手
- 结果：拿到了什么，丢了什么，关系怎么变
- 余波：谁记恨、谁心虚、谁暴露、谁不得不接着行动

落笔提醒：

- 事件不要只交代“发生了什么”，要写出人为什么这样做、被什么逼着变招
- 冲突场不要把过程一笔抹掉，至少保留关键选择和关键受阻
- 日常场也按事件处理：买药、吃饭、乘车、洗伤口，都要有行动目的和关系变化

返修提示：

- 如果情节像梗概，先把“受阻”和“变招”补出来
- 如果节奏发平，检查有没有余波；很多平，是因为事情做完就断了

## 5. 反 AI 味禁忌

- 不要把分类词库当“同义词替换表”机械套用
- 不要一段里同时写满人、景、物、事，结果每样都只沾一下
- 不要先下抽象判断，再回头找细节给它做注脚
- 不要脱离 POV 乱写作者知道但角色并未察觉的美感和意义
- 不要为了“文采”牺牲清晰行动线、冲突线和关系线

## 6. 复查问题

- 这段最主要是在写哪一类
- 读者看完后，能否记住一个具体差异点，而不是只记得“氛围不错”
- 这段细节有没有真的改变读者对人物、局势、关系或风险的理解
- 删掉最后一句判断后，这段是否仍然成立；如果不成立，说明动作和细节还不够
