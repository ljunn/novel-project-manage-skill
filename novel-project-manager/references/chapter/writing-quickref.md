# 写法速查（每章必读）

## 落笔调度

- **先定类再落笔**：先判断本段主要在写 `人 / 景 / 物 / 事`，不要一段里四样平均铺开
- **控制单位**：每段最多抓 `1 个主体 + 2 个观察维度 + 1 个变化/结果`
- **先落地再判断**：先写看得见、摸得着、会受阻的东西，再补人物判断和情绪
- **写干了先换维度**：不是继续堆形容词，而是补动作、材质、声响、气味、阻力或使用痕迹

## 分类落笔

- **写人**：优先抓肖像、言谈、动作、小习惯、关系压力，先写“哪里不对劲 / 有压迫 / 有故事”
- **写景**：优先抓天光、气象、温度、声音、气味、地表和障碍，景要服务情绪和冲突，不做壁纸
- **写物**：优先抓材质、重量、新旧、缺损、使用痕迹和归属，写它怎样被拿、藏、耗、坏
- **写事**：按“起手 -> 受阻 -> 变招 -> 结果 -> 余波”推进，不要只写结果提要

## 章节层

- **一章一件事**：核心事件不超过 2 条，多了就切章
- **章尾造压**：用未解决的张力收尾，不用感悟和总结
- **每章必须有一次实感回报**：翻盘、揭秘、升级、关系进展任选其一
- **受挫给锚点**：主角挫败后，同章或紧邻章节必须给出反击/希望的基础

## 场景层

- **状态必须转移**：每个场景开头和结尾的处境或信息量不能一样
- **高能前放慢**：爆发前插 1-2 个低强度段换气，爆发后给收尾拍，别急着跳下一幕
- **对白只做两件事**：推进冲突，或暴露性格——不用对白解释背景
- **视角纪律**：不写主角不知道的信息，不在同段切换脑内

## 语言层

- **情绪不命名，写行为**："她没说话，拿起外套走了"而不是"她很失望"
- **具体优于抽象**："刀口从肩膀切下去，皮肉往两侧翻"而不是"他受了重伤"
- **心理描写链路**：想法 → 行动 → 结果；不用旁白总结角色感受
- **规避高频 AI 套语**：以 X 为 Y、在这一刻、心中涌起、岁月/征程、结尾感悟升华、"他明白了"收尾
